# PlutoSDR Driver

Reusable PlutoSDR helper package for hardware setup, signal processing, and CW radar experiments.

## Install

```bash
pip install -e .
```

## Contents

Contains:

- **Driver** which can help with:
  - Initialize Pluto  
  - Set TX and RX  

- **Report template** in `/docs` for:
  - System-level calculations  
  - Hardware specification  
  - Software hints  

- **Script template** in:
  - `scripts/radar_projects/cw_radar_exercise.py`
